# Companies Directory - Frontlines Media (React / Vite)

This is a professional, submission-ready frontend React application for the "Companies Directory" assessment.

## Features
- React (Vite) project
- Responsive UI
- Filters: search, location, industry
- Sorting (name asc/desc)
- Pagination (configurable)
- Loading and empty states
- Mock API: static `public/companies.json`
- Easily deployable to Vercel / Netlify

## Run locally (VS Code)
1. Install Node.js (v16+).
2. Open project folder in VS Code.
3. In terminal:
   ```bash
   npm install
   npm run dev
   ```
4. Open the shown localhost URL (usually http://localhost:5173).

## Git & Submission
- Commit all files, push to a GitHub repo.
- Deploy using Vercel/Netlify (connect to GitHub).
- Record a 2-3 minute walkthrough demonstrating features (see `video_script.txt` for a suggested script).
